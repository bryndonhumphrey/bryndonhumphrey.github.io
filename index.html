Hi there! My name is Bryndon Humphrey.
I am a results-driven developer adept at thriving in collaborative work environments, finding effective yet scalable solutions, and determining which design choices provide balance between optimal user experience and performance.

In a nutshell I'm a Bay Area transplant who is originally from San Diego.
A couple hobbies of mine are; climbing, coding, cooking, DIY projects, fishing, and repairing electronics.

Currently I am a software engineer for a live event production company.
In my role I develop web applications for virtual conferences, as well as lead the technical execution of the project to completion.
As a project's technical lead, I identify the opportune solution by objectively considering developers’ feedback and connecting it with customer requirements and use cases.
My daily workload typically involves solving aforementioned problems by creating reusable components using HTML, CSS, and JavaScript frameworks such as Vue.
On an as-needed basis I parse client data using spreadsheets or python.
Once the data has been cleaned, I’ll create databases and data structures within Firebase, then integrate the data into the applications.
The typical virtual conferences that I build involve live streaming multiple sessions in concurrence, attendee Q&A, custom permissions for user groups, dedicated social media feeds, as well as gamification of the conferences to create unique and competitive experiences for attendees.

Before making the career transition to become a developer, I was a project manager and coding had been a hobby of mine since childhood.
I would always learn just enough to solve the problem in front of me – such as scripting a bot so my computer would be completing video game tasks while my AOL Instant Messenger status read
“afk @ middle school.”
At my core I’ve always been curious, and I’ve leveraged that curiosity to help me think in mindsets which promote:

  • Automating boring/tedious tasks so teams can dedicate more time and brainpower to projects that matter.
  • Collaborating with different teams to make cross-company projects run smoothly.
  • Creating memorable experiences and spreading joy.

Yelp Camp

Open app View repository
Goal

Create a yelp clone dedicated to reviewing campgrounds within a two week time frame.

Features

This app is dynamic and uses the concept of RESTful routing to interface with a database and create webpages using the information stored in the database. The data in this project is routed to a MongoDB Atlas database using the Express framework and Mongoose middleware. The Mongoose middlewear combined with EJS allows different webpage views and routes to be created using information stored in the database. User permissions were created using PassportJS and are implemented on certain features of the app, such as adding/removing campgrounds. The correct permissions will be needed to implement those kinds of changes, otherwise a dynamic error message will be shown saying why the change cannot be made.

Streamly
*Work In Progress*

View repository
Goal

Create an app using React that allows users to stream from Open Broadcast Software within a one month time frame.

Features

This app is dynamic and utilizes RESTful routing principles to interface with a database and create webpages for each particular stream. Users can use Open Broadcaster Software to sent their feed to the app's RTMP server, then external users can view their feed from the stream's webpage. Users can login using their Google login credentials though implementation of OAuth request to the Google API. The correct user permissions will be needed to edit or delete streams. This app is currently created, however it has not been deployed as continuous hosting of the RTMP server will have costs associated with it. Source code of the client side of the app can be viewed in the repository.

Funky Circles

Open app View repository
Goal

Create a fun and interactive application within a one day time frame.

Features

This app creates a circle on a canvas whenever a button on the keyboard is pressed. The circle will appear on a random spot on the canvas which has been set to the height and width of the device it is being viewed on. Depending on which key is a pressed, a sound will play and then the circle will quickly shrink away while making a hue adjustment.
